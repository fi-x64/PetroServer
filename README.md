# Getting Started

**Clone the repository**
  git clone https://github.com/fi-x64/PetroServer.git
  cd natours
  npm install
 
# Script	Description
  npm run dev:	Start dev server with nodemon <br />
  
# Run the Project
  Start the server in development mode:
    npm run dev
  By default, it will run at http://localhost:3000/
